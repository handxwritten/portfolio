Lesson 1 assignment:Semantic Web Elements

This first assignment was learning about the basics of Semantic elements. Semantic elements are a clearer way of communication between both the browser and the developer.
Whether that be due to a language barrier, or it could be people using accessibility software, etc.
It's using basic and descriptive language for describing the contents of the web elements instead of out-dated "non semantic" code.
